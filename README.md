# portfolio-ai

Personal portfolio of **Nahfid Nissar** — AI Engineer & Researcher.

**Live:** https://portfolio-ai-alpha-rose.vercel.app

## Stack
- Single-file React build (`index.html`)
- `api/chat.js` — serverless AI chat backend (OpenRouter). Requires `OPENROUTER_API_KEY`.
- Deployed on Vercel.

## Run the chat API locally
Set `OPENROUTER_API_KEY` in your environment / Vercel project settings.
